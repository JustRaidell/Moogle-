En este proyecto Moogle! he intentado hacer un buscador de texto. Para usarse simplemente copie la base de datos en la carpeta content y ejecute el codigo de la carpeta Moogle Server.
Tiene implementadas la mayoria de las funcionalidades basicas para funcionar, en varios .cs divididos en metodos y clases. 
El programa desgraciadamente no ordena las busquedas por orden de importancia (o sea la calcula pero no las ordena) y presenta problemas con el tema del snippet. Son bugs que espero poder resolver en un futuro cercano.
