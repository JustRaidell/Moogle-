En este proyecto Moogle! he intentado hacer un buscador de texto.
Para usarse siga los siguientes pasos

-Copie la base de datos (o sea los documentos en los que buscar) en la carpeta Content.
-Ejecute el proyecto.
-Aparecera una pestaña con un cuadro en el que escribir la busqueda.
-Pulse buscar y apareceran los documentos ordenados por importancia.

DATOS IMPORTANTES
-No se garantiza el correcto funcionamiento con una base de datos vacia.
-El programa busca consultas independientemente de mayusculas, minusculas y simbolos, pero es sensible a las tildes.
-En la respuesta a la busqueda salen los documentos ordenados, pero desgraciadamente, con toda su extension.O sea sale el 
texto del documento entero. Estoy trabajando en optimizar eso.
-El programa devuelve documentos sin importancia (al final, es cierto, pero lo hace) asi que es posible que 
al hacer una busqueda te interesen solo los primeros 3 resultados de los 500 que salen. Tambien trabajando.
-Una vez cargada la base de datos inicial, se recomienda no modificarla entre consultas. Si desea hacerlo debe volver a cargar el programa.
