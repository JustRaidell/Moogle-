En este proyecto Moogle! he hecho un buscador de texto.
Para usarse siga los siguientes pasos

-Copie la base de datos (o sea los documentos en los que buscar) en la carpeta Content.
-Ejecute el proyecto.
-Aparecera una pestaña con un cuadro en el que escribir la búsqueda.
-Pulse buscar y aparecerán fragmentos de dichos documentos ordenados por importancia.

DATOS IMPORTANTES
-No se garantiza el correcto funcionamiento con una base de datos vacía.
-El programa busca consultas independientemente de mayúsculas, minúsculas y símbolos, pero es sensible a las tildes.
-Si ningún documento coincide con la búsqueda el programa no devolverá nada.
-Una vez cargada la base de datos inicial, se recomienda no modificarla entre consultas. Si desea hacerlo debe volver a cargar el programa.
